Agenta now includes an MCP server for agentic authoring workflows. MCP-capable agents can connect to an Agenta project and use a curated v1 tool surface over the Agenta API to create and inspect applications, evaluators, testsets, evaluation configurations, and environments.

The v1 server is intentionally scoped: it does not start evaluation runs, deploy applications, manage billing or organizations, expose traces, or provide public multi-tenant OAuth. It uses one project-scoped Agenta API key configured on the MCP server, so hosted deployments should stay behind internal access controls.
