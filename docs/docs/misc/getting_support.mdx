We offer multiple channels to get support for Agenta:

### Self-Help Resources

- Start with this documentation for answers to common questions

### Community Support

- Join our [Slack community](https://agenta-hq.slack.com/join/shared_invite/zt-1zsafop5i-Y7~ZySbhRZvKVPV5DO_7IA) for community support:
    - Use the #05_support channel for assistance (we're active during CEST hours)
    - Share feature ideas in the #06_feature-suggestion channel
- Contribute to discussions or suggest improvements on our [Github Discussions page](https://github.com/Agenta-AI/agenta/discussions)

### Bug Reporting

- For any bugs encountered, please [file an issue on Github](https://github.com/Agenta-AI/agenta/issues)

### Cloud Support

- Pro plan cloud users receive access to a private Slack channel for direct team support

### Demos and Sales Inquiries

- For demos and more sales requests, please feel free to [book a call](https://cal.com/mahmoud-mabrouk-ogzgey/demo) with one of the founders
