Project: ClickHouse Tracing Backend

This folder tracks the plan and findings for evaluating ClickHouse as the tracing backend.

Files:
- context.md. Goals, non-goals, and scope.
- plan.md. Execution plan and milestones.
- status.md. Current progress, decisions, and blockers.
- research.md. Schema mapping notes and query analysis.
- schema.sql. Proposed ClickHouse table definition for spans.
- schema_beta.sql. Optimized schema with materialized metric columns.
- ingestion-benchmark.md. Ingestion benchmark results and methodology.
