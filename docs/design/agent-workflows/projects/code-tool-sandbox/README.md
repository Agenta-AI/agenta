# Code Tool Sandbox

This workspace answers where author-controlled code runs today, why that is a
multi-tenant security risk, and what controls Agenta needs before enabling code tools,
skills, or shell execution on a shared cloud runner.

## Files

- `explainer.md` - Plain-language walkthrough of the current execution paths.
- `research.md` - Verified codebase facts plus outside sandbox/security research.
- `proposal.md` - Architecture options and recommendation.
- `security-review.md` - Threat model, findings, requirements, and implementation proposal.
- `status.md` - Current decisions, blockers, open questions, and next steps.
