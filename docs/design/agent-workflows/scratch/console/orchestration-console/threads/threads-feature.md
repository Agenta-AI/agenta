# It's not only tasks: add conversation threads I can post into

## Summary

Added 'thread' as a first-class type: a topic with a running Summary on top and a Messages log you post into from the UI. You can open a new thread by asking a question in the UI. Threads can be promoted to tasks (carrying the summary across as the task Context). Tasks also got a message log you can post into. All of it shows in 'console pending' so the agent picks up your posts next turn.
