# Preview CI Isolation Experiment

This workspace tracks a temporary change that disables post-deploy preview tests so we can verify whether the preview environment remains stable when CI stops interacting with the live deployment after rollout.

Files:
- `context.md` - background, problem statement, goals, and non-goals
- `research.md` - workflow and log findings that motivate the experiment
- `plan.md` - the experiment plan and expected observations
- `status.md` - current implementation status and follow-up notes
