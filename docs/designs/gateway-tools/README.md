- `status.md` — Living execution log for progress, decisions, and blockers.
- `implementation-spec.md` — Technical implementation details and architecture decisions.
- `specs.md` — Functional/API specifications.
- `PR.md` — High-level scope and implementation summary for the feature.
- `api-reference.md` — API endpoint and payload reference for tools/gateway flows.
- `composio-integration.md` — Composio integration details and behavior.
- `CONNECTION_FLOWS.md` — Connection lifecycle and auth flow notes.
- `providers.md` — Provider model and assumptions.
- `TOOLS_IMPLEMENTATION.md` — Additional implementation notes.
- `summary.up-to-this-point.md` — Historical summary of work completed.
