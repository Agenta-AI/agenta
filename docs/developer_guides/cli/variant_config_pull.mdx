---
title: "agenta variant config pull"
description: "Fetches the latest configuration for an Application Variant(s)"
---

The `pull` command serves to pull the latest configuration for an App variant(s) from the backend and saves it in `<codebase_filename>.<configuration_name>`

```bash
agenta config pull
```

> Pulls the latest configuration for all Variants of the Application

or

```bash
agenta config pull <variant_name>
```

> Pulls the latest configuration for all the specified variant of the Application

<Warning> The `config pull` command can only be run in a directory where the `config.toml` generated by `agenta init` is present.</Warning>
