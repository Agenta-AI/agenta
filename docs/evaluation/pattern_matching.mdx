---
title: "Pattern Matching Evaluation"
---

- Regular Expression: Tests if output matches a given regex pattern

- String Operations:

  - Prefix Check: Verifies if output starts with a specified string

  - Suffix Check: Confirms if output ends with a specified string

Content Checks:

- Substring Presence: Verifies if output contains a specific substring

- Multiple Substring Checks:

  - Any Substring: Checks for presence of any specified substrings

  - All Substrings: Ensures all specified substrings are present
